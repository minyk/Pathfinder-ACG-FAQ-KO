# 빌드 방법

* 이 문서를 빌드하기 위해서는 [gitbook](https://github.com/GitbookIO/gitbook)이 필요합니다.

```
$ gitbook build
```

* Calibre 를 설치한 이후 [ebook-converter](http://manual.calibre-ebook.com/cli/ebook-convert.html)를 사용하여 pdf, epub 형태로 생성할 수 있습니다.

```
$ gitbook pdf
```

```
$ gitbook epub
```
